# Stock-Website — Finance Research Project

Built and maintained by Gian De Gruccio, community college student and Finance transfer candidate.

A live risk-tiered stock watchlist analyzing 15 companies through a custom scoring model using financial strength, growth, valuation, risk, and moat factors. Includes research reports, a weekly portfolio journal, and a simulated $10k portfolio with live Finnhub prices.

Live site: https://giandandivanl.github.io/Stock-Website
